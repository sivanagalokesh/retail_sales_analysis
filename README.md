# 🛍️ Retail Sales Analysis

A data analytics project that explores retail sales data to uncover trends, customer behavior, and business insights using Python and data visualization techniques.

---

## 📌 Overview

Retail businesses generate massive amounts of transactional data daily. This project analyzes retail sales data to extract **meaningful insights** that can help businesses:

* Understand sales trends
* Identify top-performing products
* Analyze customer behavior
* Improve decision-making

The goal is to transform raw data into actionable business insights through **Exploratory Data Analysis (EDA)**.

---

## 🧠 Key Features

* 📊 Exploratory Data Analysis (EDA)
* 📈 Sales trend analysis over time
* 🧍 Customer segmentation analysis
* 🛒 Product/category performance insights
* 📉 Profit and revenue analysis
* 📊 Data visualization using graphs and charts

## 🚀 Installation & Setup

### 1️⃣ Clone Repository

```bash id="yr9c6l"
git clone https://github.com/sivanagalokesh/retail_sales_analysis.git
cd retail_sales_analysis
```

---

### 2️⃣ Create Virtual Environment

```bash id="n9kvcz"
python -m venv env

# Activate
source env/bin/activate     # Linux / WSL
env\Scripts\activate        # Windows
```

---

### 3️⃣ Install Dependencies

```bash id="r9txap"
pip install -r requirements.txt
```

---

### 4️⃣ Run the Notebook

```bash id="e1b3vp"
jupyter notebook
```

Open the notebook and run all cells.
